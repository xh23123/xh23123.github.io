关于切图

一般前端会给两张图，一张是png格式的图，一张是jpg格式的图，png格式就是我们要用的图，jpg格式上面一般会标有每个东西的左上角坐标，以及图形的长和宽

我们要使用应该有一下2个css类

源类，就是整张图片，大小应该是整张图片的大小
.movie-star-ele {
background: url(‘/res/image/movie/movie_star.png’)no-repeat;
background-size: 9.2rem 15.35rem;
}

该类标出这个图形的坐标，大小
.movie-star-five {
background-position: 0 -2.9rem;
width: 6.7rem;
height: 1.2rem;
}

这样就可以显示出想要的图片了

规定行数与查看更多的实现

要规定几行，就用下面这个类就可以，就是ellipsis表示超过了就是…的表示
.three-line {
display: -webkit-box;
overflow: hidden;
text-overflow: ellipsis;
-webkit-box-orient: vertical;
-webkit-line-clamp: 3;
}
加上这个类后就可以使他只有规定的行然
如果想实现点击查看更多后显示完整的内容，只需要在点击事件中把这个类中去掉即可
