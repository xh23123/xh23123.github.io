Hexo更换皮肤

  • 日记
blog 
今天晚上很开心，没有加班，早早的就回来了，现在觉得公司的福利真的非常不错啊，早餐+晚餐+水果+按摩==

开始说正事，今天觉得这个皮肤实在是太丑了，就换了一下皮肤。

第一步：安装
$ git clone https://github.com/litten/hexo-theme-yilia.git themes/yilia
前面是路径，后面是存放的路径

第二步：配置
修改hexo根目录下的 _config.yml ： theme: **

第三步：更新
cd themes/yilia
git pull

然后再更新一下就好噜！
